## 2. 우선 도입 필요 기능

### 2.1 소셜 로그인 (긴급)
**이유**: 경쟁사 대부분 지원, 가입 전환율에 직접적 영향

**요구사항**:
- 카카오, 네이버, 구글 로그인 지원
- 기존 계정과 연동 기능
- 예상 개발 기간: 3주

### 2.2 실시간 채팅 상담 (높음)
**이유**: CS 응대 품질 차별화 요소, MegaMart와 QuickBuy의 핵심 강점

**요구사항**:
- 24시간 챗봇 기본 응대
- 영업시간 상담원 연결
- 상담 이력 저장 및 조회

### 2.3 정기 배송 서비스 (중간)
**이유**: 재구매율 향상 및 고객 락인 효과

**요구사항**:
- 배송 주기 설정 (1주/2주/1개월/2개월)
- 다음 배송 건너뛰기
- 수량 및 상품 변경
- 정기 배송 할인 적용
