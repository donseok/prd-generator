# **프로필 사진 관리 기능**

**버전**: 1.0 | **상태**: draft | **신뢰도**: 78%
**생성일**: 2026-01-22 11:57

## 1. 개요

### 배경
현재 서비스에서 사용자들의 개인화된 경험과 참여도를 높이기 위해 프로필 관리 기능의 고도화가 필요합니다. 또한 회원 등급 시스템을 통한 로열티 프로그램 강화와 고객 문의 관리 체계 개선을 통해 사용자 만족도와 재구매율을 향상시키고자 합니다.

### 목표
- 사용자가 갤러리 선택, 카메라 촬영, 크롭 기능을 통해 손쉽게 프로필 사진을 관리할 수 있도록 한다
- 회원 등급 현황과 혜택을 직관적으로 표시하여 등급 상승에 대한 동기를 부여한다
- 1:1 문의 내역 관리 기능을 제공하여 고객 지원 경험을 개선한다

### 범위
본 프로젝트는 마이페이지 내 프로필 사진 관리(갤러리 선택, 카메라 촬영, 크롭, 삭제/복원, 포맷 및 용량 제한), 회원 등급 정보 표시(등급/아이콘, 승급 필요 금액, 프로그레스 바, 혜택 목록, 산정 기준 툴팁), 1:1 문의 관리(목록 조회, 상태 필터링, 상세/답변 확인, 삭제, 추가 문의) 기능을 포함합니다.

### 범위 외
- 회원 등급 산정 로직 및 백엔드 등급 정책 변경
- 1:1 문의에 대한 관리자 답변 작성 기능 (관리자 페이지)
- 프로필 사진 외 다른 개인정보 수정 기능
- 푸시 알림 및 이메일 알림 기능
- 소셜 로그인 연동 및 계정 통합 기능

### 대상 사용자
- 서비스에 가입한 일반 회원으로 프로필을 개인화하고 싶은 사용자
- 회원 등급 혜택에 관심이 있고 등급 상승을 목표로 하는 활성 사용자
- 서비스 이용 중 문의사항이 있어 고객센터와 소통이 필요한 사용자

### 성공 지표
- 프로필 사진 설정률 30% 이상 달성
- 회원 등급 정보 페이지 조회 후 구매 전환율 10% 향상
- 1:1 문의 기능 사용자 만족도 4.0/5.0 이상
- 문의 내역 조회 기능 사용률 월간 활성 사용자 대비 20% 이상

## 2. 기능 요구사항 (FR)

### REQ-001: 갤러리에서 프로필 사진 선택
**우선순위**: MEDIUM | **신뢰도**: 82%

사용자가 디바이스 갤러리에서 프로필 사진을 선택할 수 있는 기능

**User Story**: As a 일반 사용자, I want 디바이스 갤러리에서 프로필 사진을 선택할 수 있기를, so that 원하는 사진으로 나의 프로필을 개인화할 수 있다

**Acceptance Criteria**:
- [ ] Given 사용자가 프로필 편집 화면에 있을 때, When 프로필 사진 변경 버튼을 탭하면, Then 갤러리 접근 옵션이 표시된다
- [ ] Given 갤러리 접근 권한이 없을 때, When 갤러리 선택을 시도하면, Then 권한 요청 팝업이 표시된다
- [ ] Given 갤러리 접근 권한이 허용된 상태에서, When 갤러리를 열면, Then 디바이스에 저장된 이미지 목록이 표시된다
- [ ] Given 갤러리에서 이미지를 선택했을 때, When 선택을 확인하면, Then 해당 이미지가 프로필 사진으로 적용된다
- [ ] 지원되는 이미지 형식은 JPG, PNG, HEIC이다
- [ ] 선택 가능한 이미지 최대 크기는 10MB이다
- [ ] 선택한 이미지는 프로필 사진 영역에 맞게 크롭 또는 리사이즈되어 표시된다
- [ ] Given 사용자가 갤러리 접근 권한을 거부했을 때, When 갤러리 선택을 시도하면, Then 설정에서 권한을 변경하라는 안내 메시지가 표시된다

**우선순위**: MEDIUM | **신뢰도**: 82%

**Acceptance Criteria**:
- [ ] 프로필 설정 화면에서 '카메라로 촬영' 버튼이 표시된다
- [ ] 버튼 클릭 시 카메라 접근 권한을 요청한다
- [ ] 카메라 권한이 허용되면 카메라 프리뷰 화면이 표시된다
- [ ] 촬영 버튼을 누르면 현재 프리뷰 이미지가 캡처된다
- [ ] 촬영된 사진의 미리보기와 함께 '재촬영' 및 '사용하기' 옵션이 제공된다
- [ ] 카메라 권한이 거부된 경우 권한 설정 안내 메시지가 표시된다
- [ ] 촬영된 이미지는 프로필 사진 규격(최소 200x200px)에 맞게 자동 조정된다
- [ ] 전면/후면 카메라 전환 기능이 제공된다

### REQ-003: 프로필 이미지 크롭 기능
**우선순위**: MEDIUM | **신뢰도**: 82%

선택하거나 촬영한 이미지를 원하는 영역으로 자를 수 있는 크롭 기능 제공

**User Story**: As a 사용자, I want 선택하거나 촬영한 프로필 이미지를 원하는 영역으로 크롭할 수 있기를, so that 원하는 부분만 프로필 사진으로 사용할 수 있다

**Acceptance Criteria**:
- [ ] Given 사용자가 이미지를 선택했을 때, When 크롭 화면이 표시되면, Then 이미지 위에 크롭 영역 선택 UI가 표시된다
- [ ] Given 크롭 화면에서, When 사용자가 크롭 영역을 드래그하면, Then 선택 영역의 크기와 위치가 실시간으로 조정된다
- [ ] Given 크롭 화면에서, When 사용자가 핀치 제스처를 사용하면, Then 이미지가 확대/축소된다
- [ ] Given 크롭 영역이 선택된 상태에서, When 사용자가 확인 버튼을 누르면, Then 선택된 영역만 잘린 이미지가 저장된다
- [ ] Given 크롭 화면에서, When 사용자가 취소 버튼을 누르면, Then 원본 이미지가 유지되고 이전 화면으로 돌아간다
- [ ] 크롭된 이미지의 최소 해상도는 200x200 픽셀 이상이어야 한다
- [ ] 프로필 이미지 크롭은 1:1 정사각형 비율로 고정된다

### REQ-004: 프로필 사진 삭제 및 기본 이미지 복원

**User Story**: As a 일반 사용자, I want 프로필 사진을 삭제하고 기본 이미지로 복원할 수 있기를, so that 원하지 않는 사진을 제거하고 깔끔한 기본 프로필로 되돌릴 수 있다

**Acceptance Criteria**:
- [ ] Given 사용자가 프로필 사진을 등록한 상태일 때, When 프로필 사진 삭제 버튼을 클릭하면, Then 기존 사진이 삭제되고 시스템 기본 이미지로 자동 변경된다
- [ ] Given 프로필 사진 삭제 요청 시, When 삭제가 완료되면, Then 삭제 완료 메시지가 표시된다
- [ ] Given 프로필 사진이 삭제된 후, When 프로필 페이지를 조회하면, Then 시스템 기본 이미지가 표시된다
- [ ] Given 사용자가 프로필 사진 삭제를 시도할 때, When 삭제 버튼을 클릭하면, Then 삭제 확인 다이얼로그가 표시된다
- [ ] Given 기본 이미지만 설정된 상태일 때, When 프로필을 조회하면, Then 삭제 버튼이 비활성화되거나 표시되지 않는다

- [ ] 최고 등급 도달 시 프로그레스 바 대신 최고 등급 달성 상태가 표시된다
- [ ] 프로그레스 바는 1초 이내에 로딩되어 화면에 표시된다
- [ ] 모바일과 데스크톱 환경에서 모두 프로그레스 바가 정상적으로 렌더링된다

- [ ] 현재 로그인한 회원의 등급이 강조 표시된다
- [ ] 비로그인 사용자도 등급별 혜택 목록을 조회할 수 있다
- [ ] 혜택 목록은 페이지 로드 후 2초 이내에 표시된다
- [ ] 각 혜택 항목에는 혜택명과 상세 설명이 포함된다

### REQ-010: 등급 산정 기준 툴팁 안내
**우선순위**: HIGH | **신뢰도**: 82%

회원 등급이 어떻게 산정되는지 기준을 설명하는 툴팁 제공

**User Story**: As a 회원, I want 회원 등급 산정 기준을 툴팁으로 확인할 수 있기를, so that 내 등급이 어떻게 결정되는지 이해하고 다음 등급으로 승급하기 위한 조건을 파악할 수 있다

**Acceptance Criteria**:
- [ ] 회원 등급 표시 영역에 마우스를 올리면 툴팁이 표시된다
- [ ] 툴팁에는 각 등급별 산정 기준(예: 구매 금액, 구매 횟수, 활동 기간 등)이 명시되어 있다
- [ ] 툴팁은 마우스를 영역 밖으로 이동하면 0.3초 이내에 사라진다
- [ ] 모바일 환경에서는 등급 영역을 탭하면 툴팁이 표시되고, 다른 영역 탭 시 사라진다
- [ ] 툴팁 내용은 현재 회원 등급 정책과 일치해야 한다
- [ ] 툴팁은 화면 경계를 벗어나지 않도록 위치가 자동 조정된다

### REQ-011: 1:1 문의 내역 목록 조회
**우선순위**: HIGH | **신뢰도**: 82%

사용자가 이전에 작성한 1:1 문의 내역을 최신순으로 목록 조회

**User Story**: As a 서비스 사용자, I want 내가 작성한 1:1 문의 내역을 최신순으로 목록 조회할 수 있기를, so that 이전에 문의한 내용과 답변 상태를 쉽게 확인할 수 있다

**Acceptance Criteria**:
- [ ] 사용자가 문의 내역 메뉴에 접근하면 본인이 작성한 문의만 표시된다
- [ ] 문의 목록은 작성일 기준 최신순(내림차순)으로 정렬되어 표시된다
- [ ] 각 문의 항목에는 제목, 작성일, 답변 상태(대기중/답변완료)가 표시된다
