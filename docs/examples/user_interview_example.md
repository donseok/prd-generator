## 주요 Pain Points

### 1. 검색 기능 불편
> "상품 검색할 때 원하는 결과가 잘 안 나와요. 필터도 복잡하고..."

**요구사항 도출:**
- 검색 결과 정확도 개선 필요
- 필터 UI 단순화
- 자동완성 기능 추가 요청

### 2. 주문 내역 관리
> "예전에 산 물건 다시 사려고 하면 찾기가 너무 어려워요. 재주문 버튼 같은 거 있으면 좋겠어요."

**요구사항 도출:**
- 주문 내역에서 재주문 기능
- 자주 구매하는 상품 추천
- 주문 내역 검색 기능

### 3. 리뷰 시스템
> "리뷰 쓰기가 귀찮아서 잘 안 쓰게 돼요. 근데 다른 사람 리뷰는 꼭 보거든요."

**요구사항 도출:**
- 간편 리뷰 작성 (별점만 빠르게)
- 사진 리뷰 인센티브
- 리뷰 도움됐어요 기능
