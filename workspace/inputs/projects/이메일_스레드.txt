[협의 결과]

1. 연동 방식: REST API (기존 ERP 외부연동 모듈 활용)

2. 데이터 전송
   - 일 1회 마감 배치 전송
   - 건별 전송 아닌 JSON 배열로 일괄 전송
   - 최대 1,000건 단위 페이징

3. 필수 데이터 필드
   - 계량일시 (timestamp)
   - 차량번호 (plate_number)
   - 총중량 (gross_weight)
   - 공차중량 (tare_weight)
   - 순중량 (net_weight)
   - 계량대ID (scale_id)
   - 전표번호 (receipt_number)

4. 일정
   - 시스템 연동 신청서 제출: 1/26까지
   - Client ID 발급: 신청 후 3영업일
   - Staging 환경 접근: 2/5 예정
   - 연동 테스트: 2월 중
